# Contributing to EcoCycle
## Introduction
EcoCycle is an open-source project, and we welcome contributions from the community.
## Getting Started
1. Fork the repository: `git fork https://github.com/your-username/ecocycle.git`
2. Clone the repository: `git clone https://github.com/your-username/ecocycle.git`
3. Create a new branch: `git checkout -b my-branch`
## Making Changes
1. Make your changes: `git add .`
2. Commit your changes: `git commit -m 'My changes'`
3. Push your changes: `git push origin my-branch`
## Creating a Pull Request
1. Go to the EcoCycle repository: `https://github.com/your-username/ecocycle`
2. Click on the 'Pull requests' tab
3. Click on the 'New pull request' button
4. Fill in the pull request form
5. Submit the pull request
